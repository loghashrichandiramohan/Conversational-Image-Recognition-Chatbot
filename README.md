# SCQAA-Net: Audio-Driven Multimodal AI System

## Overview
SCQAA-Net is a multimodal AI system that integrates image captioning, question generation, visual question answering (VQA), and speech synthesis into a unified pipeline. It enables real-time, context-aware interaction with images through both text and audio.

## Features
- Image Captioning using BLIP
- Context-aware Question Generation using FLAN-T5
- Visual Question Answering (VQA) using BLIP-VQA
- Text-to-Speech using Tacotron2 + WaveGlow
- Real-time interactive pipeline
- Accessibility-focused (supports visually impaired users)

## Architecture
Input Image → Feature Extraction → Caption Generation → Question Generation → VQA → Text-to-Speech → Output

## Tech Stack
- Python
- PyTorch
- Hugging Face Transformers
